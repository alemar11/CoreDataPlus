CustomTransformer

public final class CustomTransformer<T> : ValueTransformer where T : NSObject, T : NSSecureCoding

A generic ValueTransformer subclass to implement CoreData Transformable attributes.

Note

CoreData Transformable attributes are converted to and from the Data type.

    The name of the transformer. It’s used when registering the transformer using DataTransformer.register(transform:reverseTransform:). It’s composed combining the T class name and the suffix “Transformer” (i.e. if T is Object, the transformer name is ObjectTransformer).
