public protocol ManagedObjectContextNotification

CoreDataPlus

A Notification involving a NSManagedObjectContext.

Note

Since these notifications are received only when a NSManagedObject is changed manually, they aren’t triggered when executing NSBatchUpdateRequest/NSBatchDeleteRequest.
